# Message Templates — English

## Incident

```
:rotating_light: *INCIDENT: {headline}*

*Status:* {active | investigating | resolved}
*Severity:* {critical | high | medium | low}
*Affected service:* {service_name}

*Description:*
{What is happening and what is the user impact}

*Actions taken:*
• {action_1}
• {action_2}

*Next steps:*
• {next_step_1}
• {next_step_2}

*Owner:* {person}
*Last update:* {timestamp}
```

## Announcement

```
:mega: *{headline}*

{One paragraph context — why this matters}

*Changes:*
• {change_1}
• {change_2}

*Impact:*
{Who is affected and how}

*Action required:*
{What people need to do, or "No action required"}

*Effective date:* {date}
```

## Question

```
:thinking_face: *Question: {headline}*

*Context:*
{Background information needed to understand the question}

*Specific question:*
{The actual question, clearly formulated}

*Options being considered:*
1. {option_1}
2. {option_2}

*Need input from:* {audience}
*Deadline:* {date, or "When possible"}
```

## Update

```
:clipboard: *Update: {headline}*

*Progress:*
• :white_check_mark: {completed_item}
• :hourglass_flowing_sand: {in_progress_item}
• :red_circle: {blocked_item}

*Summary:*
{Brief summary of current state}

*Blockers:*
{List of blockers, or "No blockers"}
```

## Review

```
:mag: *Review: {headline}*

*What was reviewed:*
{Description of what was analyzed}

*Findings:*
1. {finding_1}
2. {finding_2}
3. {finding_3}

*Recommendations:*
• {recommendation_1}
• {recommendation_2}

*Action required:*
{What people need to do}
```
