# Message Templates — Español

## Incident

```
:rotating_light: *INCIDENTE: {headline}*

*Estado:* {activo | en investigación | resuelto}
*Severidad:* {crítica | alta | media | baja}
*Servicio afectado:* {service_name}

*Descripción:*
{What is happening and what is the user impact}

*Acciones tomadas:*
• {action_1}
• {action_2}

*Próximos pasos:*
• {next_step_1}
• {next_step_2}

*Responsable:* {person}
*Última actualización:* {timestamp}
```

## Announcement

```
:mega: *{headline}*

{One paragraph context — why this matters}

*Cambios:*
• {change_1}
• {change_2}

*Impacto:*
{Who is affected and how}

*Acción requerida:*
{What people need to do, or "No se requiere acción"}

*Fecha efectiva:* {date}
```

## Question

```
:thinking_face: *Pregunta: {headline}*

*Contexto:*
{Background information needed to understand the question}

*Pregunta específica:*
{The actual question, clearly formulated}

*Opciones que estamos considerando:*
1. {option_1}
2. {option_2}

*Necesito respuesta de:* {audience}
*Deadline:* {date, or "Cuando sea posible"}
```

## Update

```
:clipboard: *Actualización: {headline}*

*Progreso:*
• :white_check_mark: {completed_item}
• :hourglass_flowing_sand: {in_progress_item}
• :red_circle: {blocked_item}

*Resumen:*
{Brief summary of current state}

*Bloqueantes:*
{List of blockers, or "Sin bloqueantes"}
```

## Review

```
:mag: *Revisión: {headline}*

*Qué se revisó:*
{Description of what was analyzed}

*Hallazgos:*
1. {finding_1}
2. {finding_2}
3. {finding_3}

*Recomendaciones:*
• {recommendation_1}
• {recommendation_2}

*Acción requerida:*
{What people need to do}
```
